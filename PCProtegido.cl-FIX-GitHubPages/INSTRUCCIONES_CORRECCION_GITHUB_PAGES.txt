PASOS PARA CORREGIR:
1. Entrar al repositorio: redpymeia360-cloud/PCProtegido.cl
2. Eliminar los archivos mal subidos del root si existen:
   - app.js si contiene estilos CSS
   - logo.svg
   - og-pc-protegido.svg
   - download vacío
3. Subir la carpeta assets completa que viene en este ZIP.
4. Verificar que index.html quede en la raíz del repositorio.
5. Entrar a Settings > Pages.
6. En Build and deployment seleccionar:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
7. Guardar.
8. Esperar a que GitHub Pages publique el sitio.
